## 🆘 Algo não está funcionando?

### Erro de compilação
```
error: cannot find symbol
```
→ Verifique se escreveu o código corretamente (maiúsculas, pontuação)

### Erro de runtime
```
Exception in thread "main"
```
→ Leia a mensagem de erro (em inglês)
→ Use o Debugger para ver o problema

### Não consegue rodar
```
java: command not found
```
→ Java não está instalado
→ Reinstale Java 21 ou adicione ao PATH
