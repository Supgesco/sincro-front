## 🎓 **SEIS FASES DE APRENDIZADO**

### **Fase 1: Fundamentos (2 horas)**
📁 `Fase1_Fundamentos.java`

**Você aprenderá:**
- Variáveis e tipos de dados
- Operadores
- Input/Output
- If/else e loops
- Arrays básicos

**Como estudar:**
1. Abra o arquivo
2. Leia os comentários
3. Execute: `java Fase1_Fundamentos`
4. Modifique valores e teste

---

### **Fase 2: Programação Orientada a Objetos (2-3 horas)**
📁 `Fase2_OOP.java`

**Você aprenderá:**
- Criar classes
- Atributos e métodos
- Construtores
- Encapsulamento (private/public)
- Getters e setters
- Herança (extends)
- Polimorfismo
- Interfaces

**Como estudar:**
1. Leia `CHEAT_SHEET.md` (seção "Classes")
2. Abra e leia `Fase2_OOP.java`
3. Execute: `java Fase2_OOP`
4. Recrie uma classe do zero

---

### **Fase 3: Coleções (2 horas)**
📁 `Fase3_Colecoes.java`

**Você aprenderá:**
- Arrays (tamanho fixo)
- ArrayList (dinâmico) ← como listas Python
- Sets (sem duplicatas)
- HashMap (dicionários) ← como dict Python
- For-each loops
- Collections.sort()

**Como estudar:**
1. Leia `Fase3_Colecoes.java`
2. Execute: `java Fase3_Colecoes`
3. Crie uma ArrayList com seus dados

---

### **Fase 4: Exceções e I/O (2 horas)**
📁 `Fase4_ExcecoesIO.java`

**Você aprenderá:**
- Try/catch/finally
- Múltiplos catch
- Leitura de arquivos
- Escrita em arquivos
- Exceções personalizadas
- Try-with-resources

**Como estudar:**
1. Leia `Fase4_ExcecoesIO.java`
2. Execute: `java Fase4_ExcecoesIO`
3. Crie um arquivo texto e leia dele

---

### **Fase 5: Strings e Datas (2 horas)**
📁 `Fase5_StringsDatas.java`

**Você aprenderá:**
- Métodos de String
- StringBuilder
- String formatting
- LocalDate e LocalDateTime
- DateTimeFormatter
- Cálculos com datas

**Como estudar:**
1. Leia `Fase5_StringsDatas.java`
2. Execute: `java Fase5_StringsDatas`
3. Crie uma calculadora de idade

---

### **Fase 6: Padrões Avançados (2-3 horas)**
📁 `Fase6_PadroesAvancados.java`

**Você aprenderá:**
- Generics (type safety)
- Lambda expressions
- Streams API
- map, filter, reduce
- collect
- Method references

**Como estudar:**
1. Leia `Fase6_PadroesAvancados.java`
2. Execute: `java Fase6_PadroesAvancados`
3. Processe dados com streams
