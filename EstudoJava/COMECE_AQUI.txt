## 🔄 Diferenças Python ↔ Java (Quick Reference)

| Conceito | Python | Java |
|----------|--------|------|
| Tipos | Dinâmicos | Estáticos ✅ (obrigatório) |
| Print | `print()` | `System.out.println()` |
| Input | `input()` | `Scanner.nextLine()` |
| Lista | `lista = [1,2,3]` | `List<Integer> lista = new ArrayList<>()` |
| Dicionário | `d = {'a': 1}` | `Map<String, Object> m = new HashMap<>()` |
| For | `for i in range(5):` | `for (int i = 0; i < 5; i++) {` |
| If | `if x > 5:` | `if (x > 5) {` |
| Função | `def foo():` | `public void foo() {` |
| Classe | `class Foo:` | `public class Foo {` |
| Try | `try: except:` | `try { } catch () {` |
| Null | `None` | `null` |
| Lambda | `lambda x: x*2` | `x -> x*2` |

## 💡 Dicas Importantes

### 1. Não precisa decorar tudo
- Use `CHEAT_SHEET.md` como referência
- Profissionais sempre consultam documentação
- Foco em entender conceitos, não decorar sintaxe

### 2. Leia os comentários do código
- Cada arquivo tem comentários detalhados
- Compare sempre com Python
- Entenda o "porquê", não apenas o "como"

### 3. Execute e modifique
- Não basta ler, execute o código
- Mude valores
- Tente quebrar de propósito
- Use o Debugger do VS Code

### 4. Compara com Python
- Quando não entender algo em Java
- Escreva a mesma coisa em Python
- Depois veja a versão Java
- Assim a diferença fica clara

### 5. Revise frequentemente
- Revisit as fases anterior
- Relembre os conceitos
- Prática = retenção
